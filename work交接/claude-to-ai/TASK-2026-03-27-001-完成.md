### 趋势页面功能
- Top 5 Trending Skills 展示
- Top 5 Latest Skills 展示
- 完整趋势列表（支持按 category / language / stars / updated 筛选排序）
- 已正式收录的项目显示绿色 `Published` 标签并可跳转站内详情页

### 业务标准统一
按用户要求统一了业务标准：
1. 主页 `Featured / Trending / Latest` 只显示 `reviewStatus === 'reviewed'` 的 skill
2. `getSkills()` 等数据层函数增加 `isReviewedSkill()` 过滤
3. `SkillCard` 对未审核项显示 `Pending Review` 标签
4. `/trending` 页面明确说明：
   - `Published` = 已进入正式目录
   - 未标记 = 仅趋势候选，未完成业务审核/安全审核
   - 趋势可见性不等于安全审核

### 筛选标准
从 124 个趋势仓库中筛选正式收录的标准：
- 与 Claude Code / MCP / agent / coding workflow 强相关
- 信息足够完整，可归类到现有 category
- 可补齐 `whatIs / howToUse / features / useCases` 等业务字段
- 未打 `verified / reviewed` 标签（因未做安全审核）
