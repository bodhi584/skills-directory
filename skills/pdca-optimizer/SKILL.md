# PDCA Optimizer Skill

## Description
Automated PDCA (Plan-Do-Check-Act) cycle for Skills-directory optimization based on GA4 analytics data. This skill analyzes user behavior, identifies high-value content opportunities, generates new skills/content, and measures results.

## Workflow
1. **Data Analysis**: Extract GA4 metrics (page views, session duration, bounce rate)
2. **Customer Needs**: Identify top-performing categories and content gaps
3. **Content Generation**: Create new skills or optimize existing ones based on insights
4. **Deployment**: Publish optimized content to skills-directory
5. **Feedback Analysis**: Measure impact and refine approach

## Key Metrics Tracked
- Page views by category/skill
- Average session duration
- Bounce rate
- User engagement patterns

## Usage
Run daily at 09:00 UTC to generate optimization recommendations and execute content updates.
