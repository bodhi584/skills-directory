### MCP 协议

**Model Context Protocol (MCP)** 是标准化 AI 工具交互的核心协议：

- `deploystack` - MCP 服务器托管
- `haiku.rag` - RAG MCP 服务器
- `flapi` - API MCP 服务器
- `ksail` - K8s MCP 服务器

## 🚀 快速开始

### 1. 安装技能

```bash
# Claude Code
claude skill install <skill-name>

# Cursor
# 将规则添加到 .cursorrules 文件

# OpenClaw
openclaw skill install <skill-name>
```

### 2. 浏览与发现

访问 **[AntigravitySkills.org](https://www.antigravityskills.org)**：

- 🔍 搜索特定技能
- 📁 按分类浏览
- #️⃣ 按标签筛选
- ⭐ 按热度排序

### 3. 贡献你的技能

1. Fork [skills-directory](https://github.com/bodhi584/skills-directory)
2. 添加技能到 `data/` 目录
3. 提交 PR
