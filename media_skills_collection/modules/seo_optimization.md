# SEO优化模块

## 功能描述
基于媒体内容自动生成SEO友好的元数据、关键词和结构化数据。

## 使用方法
```python
# 示例代码
from seo_optimization import generate_metadata, extract_keywords

metadata = generate_metadata("content.txt")
keywords = extract_keywords("content.txt")
print(metadata)
print(keywords)
```

## SEO优化建议
- 自动生成页面标题和描述
- 提取高相关性关键词
- 生成JSON-LD结构化数据
- 优化内容可读性和搜索引擎友好性
