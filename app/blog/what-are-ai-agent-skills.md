## How to Use Skills

### For Claude Code
```bash
# Install a skill
claude skill install <skill-name>

# Use it in your project
claude <skill-command>
```

### For Cursor
Simply add the skill to your `.cursorrules` file and Cursor will automatically apply it.

### For Google Antigravity
Skills are automatically discovered from the `skills/` directory in your workspace.
